# Fedaa Albonay Full Stack Portfolio

## Run locally
```bash
npm install
npm run dev
```
Open: http://localhost:3000

## Contact form setup
Copy `.env.example` to `.env`, then add your SMTP details.

## Git commands
```bash
git init
git add .
git commit -m "Initial full stack portfolio"
git branch -M main
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```
